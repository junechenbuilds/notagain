# notagain.one

A live counter of partners on the loo right now. Tap "Mine too!" to join the count.

**He's in the bathroom. Again.**

## What is this?

A funny, lighthearted app for anyone who's ever waited for their partner to come back from the bathroom. See how many others are waiting right now, track your stats, and share the moment.

## Features

- Live global counter with regional leaderboard
- Session timer with witty commentary
- Personal stats tracking (streaks, longest session, averages)
- Shareable results with language-aware links
- Multilingual support (English, Chinese, Spanish)

## Tech Stack

- **Frontend:** Static HTML/CSS/JS on Cloudflare Pages
- **Backend:** Cloudflare Workers + Durable Objects
- **Data:** Cloudflare KV for caching, Durable Objects for counters
